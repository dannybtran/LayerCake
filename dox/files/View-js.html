View

The base class for displayable sections in LayerCake.  Extend this class to create your own custom view.

(void) draw

draw : function()

Draws the current view.  It’s probably best not to call draw on a subview, but instead call it on the rootview. e.g. this.rootview.draw(); Override draw() when extending View.
